javaファイルは複数のファイルに分かれている  
メールなどで渡すときはJAR(Java ARchive)に圧縮して渡す  
zip圧縮みたいなもの  

## パッケージを使用する
package  
パッケージの中にパッケージは入れられない  
親子関係や階層関係にはパッケージ同士はならない  

どのパッケージにも属してない  
→デフォルトパッケージ  
→無名パッケージ  

別のパッケージに所属しているパッケージを利用するには  
パッケージにに添えたクラス名を指定する  
パッケージ名をつけた完全なクラス名  
→完全限定クラス名　完全修飾クラス名　FQCN
